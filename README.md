# Omar Portfolio

A simple, semantic, and responsive HTML/CSS portfolio. Built with VS Code and deployed via Netlify Pages.

## Pages
- `index.html` — main portfolio
- `explore.html` — links, lists, and inline semantics
- `images.html` — external/local images with figure/figcaption
- `table.html` — accessible table
- `test.html`, `testNew.html` — practice pages

## Tech
- HTML5, CSS3
- Git & GitHub Pages
